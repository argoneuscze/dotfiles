# dotfiles

## Notes

- Font: Hack Nerd Font
- Cursor: Bibata-Modern-Ice, 24px

### Fedora

- `sudo dnf install adw-gtk3-theme unifont-ttf-fonts google-noto-sans-vf-fonts google-noto-serif-vf-fonts google-noto-sans-cjk-fonts google-noto-serif-cjk-fonts`
- Nerd font and cursor need to be installed manually, cursor needs to be converted to hyprcursor via `hyprcursor-util`
- Set theme and fonts using `nwg-look` for GTK applications

## Installation

- `chezmoi init`
- `chezmoi diff`
- `chezmoi apply -v`
